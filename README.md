# Social Capital: The Currency They Don't Want You to Use

**A manifesto against going it alone.**

A ~30K word polemic arguing that strategic altruism — generous tit-for-tat in everyday life — is the best strategy for building a rich, resilient life. No hustle required. No empire. Just deliberate generosity, ruthless pruning, and the oldest currency humans have: social capital.
