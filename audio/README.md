# Audiobook Recordings

Drop chapter recordings here. Suggested naming:

```
ch01-grind-trap.mp3
ch02-nobody-self-made.mp3
ch03-isolation.mp3
...
ch20-the-manifesto.mp3
```

## Recording tips
- ~8-10 min per chapter (~1,400 words)
- Use `---` section breaks as natural pauses
- Record in a quiet room, close to the mic
- Audacity (free) works great for recording + editing
